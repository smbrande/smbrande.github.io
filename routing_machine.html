Automated Wire Harness Machine | January 2021 - Current

I was selected by Northrope Grumman and the University of Michigan to join a Multidisciplinary design team where I am building an automated wire harness machine for spacecraft. Although this project is a work in progress, here are some examples of equipment I’ve prototyped so far.

The machine will be required to route a wire harness without excess wire left behind to be cut by a human operator. In the picture above is a extruder I designed to provide feedback to a set of rollers that move wire. The extruder provides feedback via a custom full bridge load cell connected to an arduino uno. This feedback controls the rate at which wire is extruded, this function keeps the wire tight significantly reducing the axcess wire at the end of routing.

Since we are building a wire harness with multiple wires routed in differentiating paths, I designed a wire switcher pictured above. This wire switcher selects up to five different gauge wires to be routed consecutively. Through a G code sender called Chilipeppr I was able to test the switcher using a TinyG CNC motor controller.

I designed and built a CNC machine to hold all the components of our wire harness machine. The machine is belt driven and operates smoothly with no backlash or binding. I’m designing this machine with a high attention to detail by incorporating small features like adjustable tension brackets for the belts. For testing chilipeppr and TingG are used.

In addition to routing, and switching wire, we also want to cut and strip wire. The picture above shows a cutstrip system I designed despite using an industrial system instead.
